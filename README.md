# Marketing Funnel & Conversion Performance Analysis

## Project Overview
This project focuses on analyzing the marketing funnel and customer conversion performance using Power BI.  
The dashboard helps track user movement through different funnel stages, identify conversion drop-offs, and evaluate overall marketing effectiveness.

## Objectives
- Analyze customer journey through the marketing funnel
- Measure conversion rates across stages
- Identify funnel drop-off points
- Track campaign performance using KPIs
- Create an interactive Power BI dashboard for business insights

## Dataset Information
The dataset includes:
- Leads generated
- Website visits
- Sign-ups
- Purchases
- Conversion rates
- Campaign performance metrics
- Revenue and order values

## 🛠 Tools & Technologies Used
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization
- DAX Measures

## 📊 Dashboard Features
- KPI Cards for key metrics
- Funnel Chart for conversion stages
- Donut & Bar Charts
- Revenue Analysis
- Conversion Performance Tracking
- Interactive Filters & Slicers

## Key Insights
- Identified major customer drop-off stages
- Compared conversion performance across campaigns
- Analyzed revenue contribution and order values
- Improved understanding of marketing effectiveness

## Data Cleaning Process
- Removed blank rows and duplicates
- Checked and corrected data types
- Handled missing values
- Standardized column names
- Verified data consistency

## Project Outcome
Built an interactive and insightful dashboard to help understand customer behavior, optimize marketing strategies, and improve conversion performance.
